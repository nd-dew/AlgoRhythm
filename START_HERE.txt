🎯 STEP-BY-STEP SETUP

[ ] Step 1: Get a Gemini API Key
    → Visit: https://makersuite.google.com/app/apikey
    → Create a new key
    → Copy it

[ ] Step 2: Set your API key
    → Run: export GEMINI_API_KEY='your-key-here'
    → Or add to ~/.bashrc for permanent setup

[ ] Step 3: Install dependencies
    → Run: ./quickstart.sh
    → Or manually: pip install -r requirements.txt && npm install

[ ] Step 4: Start the server (Terminal 1)
    → Run: npm start
    → Keep this terminal open

[ ] Step 5: Start the AI agent (Terminal 2)
    → Run: python3 strudel_ai_agent.py
    → Keep this terminal open

[ ] Step 6: Open browser
    → Go to: http://localhost:3000
    → You should see the Strudel interface

[ ] Step 7: Create your first track!
    → In Terminal 2, type:
      "create a minimal techno track with kicks and hi-hats"
    → Watch the magic happen! 🎵

❓ TROUBLESHOOTING

Problem: "Gemini API key not found"
Solution: export GEMINI_API_KEY='your-key'

Problem: "Server not running"
Solution: Make sure npm start is running in Terminal 1

Problem: "No sound"
Solution: Click Play ▶️ button in browser
